## 📖 Sobre

O **Copa 2026** é uma aplicação web desenvolvida para acompanhar todos os detalhes da Copa do Mundo FIFA 2026 de forma rápida, moderna e responsiva.

O projeto foi criado utilizando apenas tecnologias front-end, proporcionando uma navegação leve e compatível com qualquer dispositivo.

## ✨ Funcionalidades

- 🏆 Tabela da competição
- ⚽ Jogos da fase de grupos
- 📅 Calendário completo
- 📊 Classificação atualizada
- 🎯 Mata-mata
- 📱 Layout responsivo
- 🌙 Interface moderna
- 🚀 Carregamento rápido

## 🚀 Como executar

Clone o repositório:

```bash
git clone https://github.com/kingmoldesofc-alt/copa-2026.git
```

Entre na pasta:

```bash
cd copa-2026
```

Abra o arquivo:

```text
index.html
```

ou utilize uma extensão como **Live Server** no VS Code.
